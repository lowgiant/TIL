# Spanning Tree
## 1) 특징
- 그래프를 구성하는 모든 간선과 정점으로 만들어진 트리
- 그래프에서 사이클이 형성되지 않고 트리를 구성
## 2) 최소 비용 신장트리
- 간선이 별로 없을 경우 크루스칼 알고리즘이 빠름
- 밀집그래프인 경우 프림 알고리즘이 빠름
- 종류
  1. Kruskal 알고리즘
     - 가장 적은 비용으로 모든 노드를 연결해 사용한 알고리즘
     - 시간복잡도: `O(nlogn)`
  2. Prim 알고리즘
     - 주어진 정점에서 가장 가중치가 적은 간선 선택
     - 선택된 간선으로 두 정점을 연결하고 사이클이 생기지 않게 하면됨
     - 시간복잡도: `n^2`
